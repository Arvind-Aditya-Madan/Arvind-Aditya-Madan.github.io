UNC Baseball Attendance Prediction

End-to-end attendance forecasting framework and Streamlit decision-support app for UNC baseball home games. The project combined historical attendance, schedule context, opponent strength, weather, academic calendar timing, Google Trends, promotions, and team performance data to help support staffing, concessions, marketing, and game-day operations.

  • Built with chronological validation, weighted rolling cross-validation, feature tiers, Optuna tuning, and backward feature elimination.
  • Supported both preseason forecasts and two-day-ahead update forecasts for operational planning.
  • My work focused on data cleaning, feature engineering, master dataset integration, weighted CV, preseason workflow adaptation, and weather/attendance features.

MLB · Machine learning

Pitch Forecaster

Collaborative machine learning project predicting the next pitch type in an MLB plate appearance using only pre-pitch information. The project used Statcast/Baseball Savant data, game context, lagged pitch-sequence features, pitcher arsenals, batter tendencies, and multiclass classification models.

  • Compared logistic regression, random forest, neural network, and ensemble approaches.
  • Final ensemble reached 46% accuracy and a 0.42 macro F1 score across nine pitch classes.

ACC basketball · Spread prediction

ACC Spread Predictor

Competition project to predict the home-away point spread for remaining ACC men’s basketball games starting February 6 using only information available before that date. I built first-half team features, tested RAW vs. DIFF feature representations, and trained rolling-season models that mimic a deployable forecasting setup.

  • Random Forest with DIFF-based features performed best, reaching mean MAE around 9.19 points.
  • Used Optuna tuning, AP Poll features, venue/rest effects, ACC-only performance, and split conformal prediction intervals.
  • Placed 6th in the Triangle Sports Analytics Competition.

Data ethics · Sports gambling

When Data Bets Against You

Research paper on the ethics of AI-driven sports betting platforms. The paper examines how behavioral data, personalization, promotions, and real-time targeting can either support harm prevention or amplify risk when engagement is prioritized over user well-being.

  • Focused on autonomy, privacy, consumer protection, accountability, and governance.
  • Argued for transparency, independent audits, protective interventions, and stricter limits on exploitative personalization.

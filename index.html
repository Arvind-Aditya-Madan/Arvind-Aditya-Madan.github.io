DATA 780 Machine Learning • MLB pitch-level data

Pitch Forecaster: MLB Pitch Type Prediction

Built a multiclass machine learning pipeline to predict the next pitch type in an MLB plate appearance using pre-pitch context, sequencing features, pitcher and batter tendencies, handedness, and game state.

Compared logistic regression, random forest, neural network, and ensemble models. The final ensemble achieved 46% accuracy and 0.42 macro F1 while improving minority pitch-type performance.
